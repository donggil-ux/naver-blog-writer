# 🍃 네이버 블로그 AI 작성기

네이버 블로그 맛집·문화생활·일상 포스팅을 자동으로 생성해주는 AI 웹앱.

## 기능

- 🍽 **음식** — 맛집/카페 방문 후기 포스팅
- 🎨 **문화생활** — 전시/공연/팝업 관람 후기 포스팅
- 📅 **일상** — 제품 리뷰/데일리 포스팅
- 🔑 트렌드 키워드 자동 수집 (SEO 최적화)
- 📝 내 글 스타일 학습 후 동일 말투로 작성
- 📸 사진 첨부 지원

## 로컬 실행

```bash
npm install
npm run dev
```

## 기술 스택

- React + Vite
- Anthropic Claude API (`claude-sonnet-4-20250514`)
- Web Search Tool (`web_search_20250305`)
